How to read the plot

Origins and destinations are represented by the circleâ€™s segments. Each state/county is assigned a color. Flows have the same color as their origin and the width indicates their size. The direction of the flow is also shown by the gap between flow and state/county: small gap indicates origin; large gap indicates destination. Click on a segment to switch between county and state view. Hover over segments or chords to for more details.

Switch between time periods using number keys (1,2); zoom in/out with Ctrl and +/-

Data Source: IRS-SOI County to County migration
